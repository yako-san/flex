## Convention de nommage

- Noms longs en français descriptifs (pas de slugs).
- Pas d'accent dans les filenames (URL/path issues) — utiliser `beta plus`
  pour `β+`, `velo` pour `vélo`, etc.
- `[[Lien]]` dans le markdown ; sur GitHub ça ne render pas en cliquable
  mais c'est facile à grep.
